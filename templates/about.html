About Pysible

Pysible is a project created by me, Artur Vaskovets, for a final work named PASIR. The project was inspired by Phansible, but has no goal to copy it. The main goal of Pysible is to provide a flexible and powerful tool that can make the life of Ansible administrators a little bit easier.

Technologies

Another goal of the project is to use Python and Flask as a base. Honestly, I have never used Python for Web Development before. So, it was a great experience. Here is a full list of technologies used in the project:

  • Python 3
  • Flask
  • Jinja2 (included by default)
  • WTForms
  • SQLAlchemy
  • Bootstrap 4
  • Flask Extensions
    • Flask-CLI, Click (included by default)
    • Flask-Bootstrap4
    • Flask-Login
    • Flask-SQLAlchemy
    • Flask-WTF
